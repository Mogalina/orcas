# Changelog

All notable changes to Orcas will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-01-15

### Added
- Initial release
- Natural language to bash command conversion
- Local LLM support (Phi-2, TinyLlama, Llama 3.2)
- Security validation and command blocking
- Interactive and single-command modes
- Sudo privilege handling with confirmation
- Dry-run mode
- Configuration system
- Automatic model downloader
- Documentation
